Slut dig til dine Dyson Cool (infrarød) eller Dyson Link (Wi-Fi) ventilatorer med Homey for det sejeste smarte hjem.

OPRET
Opret Flows for at tænde din Dyson ventilator, eller sluk den automatisk, når du er gået i seng.

MÅL
Mål dit hjems miljø og se temperaturen og luftfugtigheden i dit rum.

KONTROLLER
Kontroller din Dyson ventilator fra Homey-appen hvor som helst i verden.

* Alle Wi-Fi ventilatorer burde fungere uden problemer.
* Hvis din ventilator opfører sig mærkeligt, så giv Homey Support besked, så de kan hjælpe dig.
