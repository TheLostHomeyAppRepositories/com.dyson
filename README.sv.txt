Anslut din Dyson Cool (Infraröd) eller Dyson Link (Wi-Fi) fläkt med Homey för det coolaste smarta hemmet.

SKAPA
Skapa Flows för att sätta på din Dyson fläkt, eller stäng av den automatiskt efter att du gått och lagt dig.

MÄT
Mät hemmets miljö och se temperaturen och luftfuktigheten i ditt rum.

KONTROLLERA
Kontrollera din Dyson fläkt från Homey-appen var du än är i världen.

* Alla Wi-Fi fläktar borde fungera utan problem.
* Om din fläkt beter sig konstigt, vänligen meddela Homey Support så att de kan hjälpa dig.
